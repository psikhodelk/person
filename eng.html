4. Категория: Grammar Clues (Грамматические подсказки)

Term (Left Side)Clue / Rule (Right Side)
Currently / At the momentUse Present Continuous (am/is/are + ing)
Usually / Every yearUse Present Simple (do/does)
Yesterday / In 2022Use Past Simple (2nd form / -ed)
So far / Already / NeverUse Present Perfect (have/has + 3rd form)
If + Present Simple......then use WILL (First Conditional)
If + Past Simple......then use WOULD (Second Conditional)
